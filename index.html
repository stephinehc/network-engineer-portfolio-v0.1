About Me // who I am

Hi, I'm Stephine 👋 — an Engineering College Instructor and licensed Electronics Engineer, also a Certified Cisco Instructor for ITN (Introduction to Networks), SRWE (Switching, Routing, and Wireless Essentials), and ENSA (Enterprise Networking, Security, and Automation) under the Cisco Networking Academy curriculum.

Years of teaching and lab work have kept me consistently hands-on with Cisco routers and switches. This portfolio is where I bring that classroom experience into my own structured roadmap and project builds — tracking every step of the journey in this repository.

Beyond networking, I'm currently exploring cybersecurity and data analytics — natural next steps that build on the same infrastructure foundation: securing what I build, and making sense of the data that moves across it.

Networking Roadmap // a guide for learners

This 9-stage path is what I used to build the knowledge behind my projects — kept here as a reference for anyone else working toward network engineering. If you're reviewing my hands-on skills, see the Skills Matrix and Projects instead.

1. Network Design Fundamentals

Hierarchical models, redundancy planning, translating requirements into topology.

View on GitHub →

2. LAN Technologies

Ethernet standards, cabling, VLANs and trunking (802.1Q).

View on GitHub →

3. Switching Technologies

STP/RSTP, EtherChannel/LACP, inter-VLAN routing.

View on GitHub →

4. Routing Technologies

Static routing, RIP, OSPF, EIGRP, redistribution and summarization.

View on GitHub →

5. Network Services

DHCP, DNS, NAT/PAT, NTP, and troubleshooting each.

View on GitHub →

6. Network Security

ACLs, port security, AAA, VPN fundamentals, device hardening.

View on GitHub →

7. High Availability

HSRP/VRRP/GLBP, redundant links, designing for zero single points of failure.

View on GitHub →

8. Secure WAN

Site-to-site and remote-access VPNs, IPsec, connecting branches securely.

View on GitHub →

9. Enterprise Campus Network Concepts

Campus architecture, QoS basics, wireless integration, designing for scale.

View on GitHub →
