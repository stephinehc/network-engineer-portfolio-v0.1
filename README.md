# 🌐 Network Engineer Portfolio

Welcome — I'm an Engineering College Instructor, a licensed Electronics Engineer, and a Certified
Cisco Instructor (ITN, SRWE, ENSA), with hands-on exposure to Cisco routers and switches through
years of teaching. This repository is my portfolio: a reference for interviewers to review the
network builds I've completed, backed by a [Skills Matrix](./skills-matrix/README.md) of what I
already know. The [Networking Roadmap](./networking-roadmap/README.md) is included as a study
guide for anyone working toward network engineering — it's a resource, not a progress tracker.
